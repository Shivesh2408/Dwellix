# Dwellix Design Bible v1.0

> **Project:** Dwellix\
> **Tagline:** *Your Home. Smarter Every Day.*\
> **Status:** Living Document

------------------------------------------------------------------------

# 1. Purpose

This document defines the official design, UI, UX, branding, and
development standards for Dwellix.

Every screen, component, API-facing UI, and AI-generated output **must
follow this document**.

------------------------------------------------------------------------

# 2. Brand Personality

-   Premium
-   Minimal
-   Intelligent
-   Trustworthy
-   Modern
-   Spacious
-   Professional

------------------------------------------------------------------------

# 3. Design Inspiration

-   Apple
-   Stripe
-   Linear
-   Vercel
-   Notion

Goal: Build a product that looks like a funded SaaS startup, not a
college project.

# 6. UI Principles

-   Large whitespace
-   Clean layouts
-   Consistent spacing
-   Soft shadows
-   Rounded corners
-   Responsive-first
-   Accessibility by default

------------------------------------------------------------------------

# 7. Components

## Buttons

-   Primary
-   Secondary
-   Ghost
-   Danger

## Cards

-   16px radius
-   Soft shadow

## Inputs

-   Clean validation
-   Accessible

## Icons

-   Lucide Icons

------------------------------------------------------------------------

# 8. Animations

Only use: - Fade - Slide - Scale

Avoid flashy animations.

------------------------------------------------------------------------

# 9. Images

Every major section must include meaningful visuals.

Never use placeholders.

Whenever an illustration is required, generate it using a dedicated AI
prompt.

------------------------------------------------------------------------

# 10. Landing Page Philosophy

Sell the vision before the features.

The homepage should tell a story and build trust.

------------------------------------------------------------------------

# 11. Coding Standards

-   TypeScript everywhere
-   Reusable components
-   Clean folder structure
-   No hardcoded secrets
-   Production-ready code

------------------------------------------------------------------------

# 12. AI Rule

Every AI coding prompt begins with:

> **Follow Dwellix Design Bible v1.0**
